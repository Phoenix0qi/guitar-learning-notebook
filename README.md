# 🎸动机 Motivation
这个项目是一份吉他学习成长笔记，记录从零碎练习到系统提升的整个过程。作为一个热爱吉他的普通人，我希望通过“记录+总结+反馈”的方式，提升自己的演奏技能，同时也能给同样在学习路上的人提供参考。

在练习过程中，我发现单一的练习缺乏目标感，容易重复低效，进步也难以可视化。于是我决定：

以每周计划驱动练习节奏；

定期输出学习内容与成果作品；
